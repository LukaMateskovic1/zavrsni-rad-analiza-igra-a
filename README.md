# zavrsni-rad-analiza-igrača
Web aplikacija za analizu i usporedbu performansi nogometaša u top 5 europskih liga

## Opis projekta
Cilj projekta je razvoj web aplikacije koja omogućuje analizu i usporedbu statističkih performansi nogometaša na temelju javno dostupnih podataka. Aplikacija je namijenjena korisnicima koji žele jednostavno pregledavati i uspoređivati igrače prema različitim kriterijima.

Glavne funkcionalnosti uključuju:
- filtriranje igrača prema ligi, klubu, poziciji i sezoni
- prikaz ključnih statističkih pokazatelja (npr. golovi, asistencije, odigrane minute)
- usporedbu jednog ili više igrača putem grafičkih prikaza

## Tehnologije
Planirane tehnologije za razvoj sustava:
- Frontend: React, HTML, CSS, Javascript
- Backend: Python
- Baza podataka: još se razmišljam

## Podaci
Podaci korišteni u aplikaciji preuzimaju se iz javno dostupnih izvora, uključujući:
- FBref
- Kaggle
- Transfermarkt

## Struktura projekta
Projekt je organiziran u sljedeće cjeline:
- `frontend/` – implementacija korisničkog sučelja
- `backend/` – serverska logika i obrada podataka
- `data/` – skupovi podataka korišteni u aplikaciji
- `docs/` – dokumentacija i materijali vezani uz izradu završnog rada

## Status projekta
Projekt se trenutno nalazi u fazi planiranja i inicijalnog razvoja.
